# offgridnode-oss

Open-source compliance repository for OffGridNode products.

This repo contains the source code for GPL-licensed software distributed on OffGridNode devices, as required by the GNU General Public License.

## Structure

```
kiwix-tools/   # Source for kiwix-serve and related tools (GPLv3)
kernel/        # Linux kernel config and patches, if modified (GPLv2)
```

## For Customers

If you purchased an OffGridNode device and would like the complete corresponding source code for any GPL-licensed component, you may:

1. Download it from this repository
2. Email us at oss@offgridnode.com with your order number

We will provide the source on a physical medium for the cost of distribution, as permitted by the GPL.
